This directory contains tiny, UVL examples for `mech-uvl`.

The files are organised by CLI command first:

- `parse/ok`
- `parse/fail`
- `check/ok`
- `check/fail`
- `check/semantic-variant`
- `levels/ok`
- `levels/fail`

`check/semantic-variant` is for examples whose expected outcome depends on the selected semantic variant rather than being intrinsically accepted or rejected.

Each `.uvl` records the expected CLI behaviour in comments.
